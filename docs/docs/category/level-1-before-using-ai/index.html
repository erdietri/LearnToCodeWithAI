Level 1: Before Using AI

Contains all modules of the first level of this course.

📄️ Think Like A Developer

How many coding or Python courses have you taken that taught you how to actually think like a developer, or programmatically? ...I'd be willing to bet none - which is a problem because coding is a problem-solving skill, not something to be memorized. This video breaks down why so many people struggle to learn to code and how to approach the course going forward.

📄️ Initial Environment Setup

Where do network automators write their code? This video explains the integrated developer environment (IDE) and how coders of all kinds can get started with it.

📄️ Data Types & Variables

Computers are dumb, so we must tell them what information to remember and what exactly that information is. After all, coding is really just having your computer remember information (i.e. that a switch is in active mode) and then creating logical actions on that information (i.e. having that switch failover to passive mode). This video explains how we "store" information in something called a variable.

📄️ Functions (Part 1) - Overview

Functions are where all of the actions, or logic, happens! In this video, I essentially teach you how to make your script actually do stuff in Python. Python provides some built-in logic or functionality that you'll use, but you'll also learn the basics of how to create your own.

📄️ Functions (Part 2) - If/Else & Loops

Python also has built-in conditions that we can use when creating "actions" or functions in our code. Two important ones are the concepts of if/else statements and loops, and you'll use them all the time in a variety of scenarios... let's learn more!

📄️ Troubleshoot Your Code

As you know, everything breaks...including our code. In this video, I expose you to the concept of troubleshooting your code with the debugger that is built into your developer environment. Don't focus too much on how comfortable you feel with the debugger just yet; simply focus on practicing using it to walk through and understand your code or to troubleshoot errors. (Trust me, you'll get to use AI for some of this later!)

📄️ Understanding Libraries

As coders, we don't need to code everything from scratch. In fact, at times, doing so is not a good use of our time. Many coders have come before you to write code to help solve various use cases and have made them publicly available to you. This code is packaged in what we call a "library," and you can install it in your project so that you can use logic they have already created.

📄️ Git Essentials

Coding doesn't happen in a vaccum. We need to be able to track changes in our code over time and to share our code with others. This video explains how to do that by leveraging versional control software (git) and a remote repository (GitHub).
