Where do network automators write their code? This video explains the integrated developer environment (IDE) and how coders of all kinds can get started with it.

## Practice what you preach 

1. Download and install VS Code: https://code.visualstudio.com/download

2. Install the Python extension: https://code.visualstudio.com/docs/languages/python#_install-python-and-the-python-extension

3. Download and install Python3 for your OS: https://www.python.org/downloads/

4. Select the Python interpreter: https://code.visualstudio.com/docs/languages/python#_install-python-and-the-python-extension

## Have your mentor ask you this...

1. First, did the student install their developer environment, Python, and the Python extension? If not, talk through and troubleshoot any obstacles.

2. After watching this video, can you think of any reasons off the bat that you would want to code in an IDE rather than notepad or your terminal? Discuss.

3. Ask the student to create a new Python file, write something (anything) in it, and then run it.

4. Let's pretend you've written a new script that you'd like to run. You run the file, only to see an error stating "No developer tools found" in the terminal. How would you troubleshoot this?

5. Ask the student to create a second Python terminal in their IDE. (Help them if they forget.)

Bravo!! You've taken the first step along your journey!
